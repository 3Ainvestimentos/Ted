# **App Name**: Ted 1.0

## Core Features:

- Risk Assessment: AI-powered analysis of project data across Focalboard boards, to identify risks and delays; uses a tool for pattern recognition to flag critical issues.
- Executive Summaries: Automated report generation summarizing progress and key metrics, to deliver executive summaries for management reviews.
- Meeting Automation: Assisted meeting minutes creation; converts updates on cards into structured notes.
- Central Dashboard: Dashboard presenting project status and insights, from each strategic initiative and comittee.
- Strategic Initiative Dossier: Card view providing access to a live dossier for the strategic initiative.
- User Authentication and roles: Role-based access control, where different users or roles, like PMOs, or leaders can visualize, contribute, comment, and update initiative status.

## Style Guidelines:

- Background (Fundo Principal): hsl(0 0% 100%) (Branco puro)
- Foreground (Texto Principal): hsl(220 10% 23%) (Cinza escuro levemente azulado, para alta legibilidade)
- Card/Superfícies Elevadas: hsl(0 0% 100%) (Branco puro, geralmente com bordas sutis hsl(220 15% 88%))
- Primary (Cor de Destaque Principal): hsl(214 79% 51%) (Azul vibrante, usado para botões primários, links importantes, ícones ativos)
- Primary Foreground (Texto sobre Cor Primária): hsl(0 0% 100%) (Branco puro, para contraste com a cor primária)
- Secondary (Elementos de Suporte/Menor Destaque): hsl(220 16% 94%) (Cinza claro, usado para fundos de input, botões secundários)
- Muted (Texto Suave, Divisores, Bordas): hsl(220 15% 90%) (Cinza ainda mais claro)
- Muted Foreground (Texto sobre Fundo Muted): hsl(220 10% 45%) (Cinza médio)
- Accent (Ênfase Sutil, Hovers, Itens Ativos/Selecionados): hsl(220 16% 92%) (Cinza claro, similar à secundária, mas usada para interações)
- Accent Foreground (Texto sobre Cor Accent): hsl(220 10% 20%) (Cinza escuro)
- Destructive (Ações Destrutivas, Erros): hsl(0 75% 60%) (Vermelho claro e vibrante)
- Destructive Foreground (Texto sobre Cor Destrutiva): hsl(0 0% 100%) (Branco puro)
- Border (Bordas Padrão): hsl(220 15% 88%) (Cinza claro e sutil)
- Input (Fundo de Campos de Input): hsl(220 16% 94%) (Cinza claro, similar à secundária)
- Ring (Anel de Foco): hsl(214 79% 61%) (Um tom de azul levemente mais claro que o primário)
- Fonte Principal: 'Archivo' (sans-serif). Esta fonte é versátil e usada para todo o texto da interface, incluindo corpo, títulos e elementos de navegação.
- Pesos: A fonte 'Archivo' é utilizada em diversos pesos (ex: 300 Light, 400 Regular, 500 Medium, 600 SemiBold, 700 Bold) para criar hierarquia visual.
- Estilo: Limpo, moderno e altamente legível.
- Use minimalistic icons to represent project statuses, risks, and priorities.
- Card-based layout inspired by Focalboard to allow ease of use.
